tartaruga è una libreria di supporto all'insegnamento di Programmazione del corso di studi in Informatica musicale dell'Università degli Studi di Milano. Si basa sul pacchetto javax.swing e fornisce alcune semplici primitive per la turtle graphics.

Usare tartaruga è molto semplice, è sufficiente istanziare un oggetto di classe {@link it.unimi.di.tartaruga.Tartaruga}, esso rappresenta una tartaruga che inizialmente si trova al centro di uno spazio di 400 x 400 unità ed è orientata verso destra; le istruzioni che si possono impartire alla tartaruga sono:

  • {@linkplain it.unimi.di.tartaruga.Tartaruga#destra(int) destra}: che fa girare a destra la tartaruga,
  • {@linkplain it.unimi.di.tartaruga.Tartaruga#sinistra(int) sinistra}: che fa girare a sinistra la tartaruga,
  • {@linkplain it.unimi.di.tartaruga.Tartaruga#avanti(int) avanti}: che fa avanzare la tartaruga (disegnando),
  • {@linkplain it.unimi.di.tartaruga.Tartaruga#sposta(int) sposta}: che fa spostare la tartaruga (senza disegnare);
gli spostamenti sono espressi in unità, mentre le rotazioni in gradi.

Ad esempio, per far disegnare un quadrato di lato 100 alla tartaruga è sufficiente implementare la classe:

public class Quadrato {
	public static void main( String[] args ) {
		Tartartuga t = new Tartaruga();
		for ( int i = 0; i < 4; i++ ) {
			t.avanti( 100 );
			t.destra( 90 );
		}
	}
}

Il pacchetto {@link it.unimi.di.tartaruga.esempi} contiene altri esempi come ad esempio il fiocco di neve di Koch e il triangolo di Sierpinski.
